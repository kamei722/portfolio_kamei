はじめに

ご覧いただきありがとうございます。ここは私、亀井が制作したゲーム作品を紹介するページです。

短期間ではあるものの、私のこれまでの勉学やチームワーク経験の蓄積が表現できているものだと思います。ユーザーテストを友人や同期と重ね、フィードバックを反映させる過程で作品を磨いてきました。イラスト制作やコーディングの効率化には生成AIを利用しています。

これらの作品を通して、私の思考プロセスを感じていただければ幸いです。

ポリシー

2作品を通して私が重視したポイントは以下の2点です。

  • 引き算の美学　 - 余計な要素を削ぎ落とした、シンプルな形を目指す
  • 少しばかりの新規性 　- 既存の枠組みに小さな新しさを

これらのバランスを突き詰め、シンプルながらも新鮮さを感じられるコンセプトの実現を目指しました。

ゲーム概要

変化する数字の足場の中で鍵を集める、シンプルでレトロなアクションゲームです。

モチベーション

私の思うアクションゲーム最大の魅力である「頭での思考と記憶」が「手先の操作」と連動するハーモニーを、シンプルな形で実現することを目指して開発しました。プレイヤーが考えた道筋がうまく表現され達成感を味わえるゲーム体験の創出を目標としました。

こだわりポイント

シンプルにまとまったゲーム性

アクションはゲームの王道である上下左右の移動に、変化する数字の足場を組み合わせました。画面上の要素は少ないけれども、その変化により複雑さを高め、直感的に理解でき、かつ集中力を要する構造を実現しています。

ステージデザイン

規則的な数列モチーフとアクションの面白さを両立する設計を目指しました。構成としても多くのパターンを試し、難易度の自然な上昇と新たな発見を両立させ、プレイヤーが継続的に刺激を受けられるレベルデザインを追求しました。

使用技術

  • Python

本作は全てPythonで実装しています。私の主要言語がPythonだったため、Pygameを活用してゲーム開発の基礎を学びました。キャラクターの移動制御、衝突判定、ステージデータの読み込みなど、ゲーム開発の基本要素を1から実装することで、ゲーム制作の根幹を理解する貴重な経験となりました。

ゲーム概要

ランダムに決定される相性で、その相性を予測しながら目標得点を目指す、もちをモチーフとした戦略型じゃんけんゲームです。

モチベーション

様々なゲームをプレイする中で、「運とスキルのバランス」や「駆け引き」がゲームの楽しさに及ぼす影響に関心を持ち、自分なりに、シンプルで戦略性とギャンブル要素を兼ね備えたゲームというお題を設定して、このゲームを制作しました。

こだわりポイント

ゲーム性とデザインのバランス

ゲームシステムを先に考案した後、それをどうプレイヤーに届けるか検討しました。「ランダムに決まる概念の表現」「同じ手ばかり選ぶのを防ぐには」といった課題に対して、「属性の概念が不要で親しみやすい」「団子という形での履歴管理」がマッチしていると考えモチーフを「もち」に定め、「おばあちゃんちで遊ぶ」ような温かみのある雰囲気のゲームに仕上げました。

選択に意味を持たせる工夫

直近3回で異なる種類の餅を選ぶと「だんごボーナス」が発生するルールを導入しました。「団子の色を変える」という視覚的な目標、プレイヤーが多様な選択肢を試すような誘導、相手の選択を予測する駆け引きの要素という3つ役割を持つ要素になりました。

カスタマイズ性

メニュー画面で様々なゲーム設定を変更できるようにし、自分の好みや腕前に合わせてプレイできるようにしました。より幅広いプレイヤーに向けた工夫になっています。

使用技術

  • Unity (2022.3.42f1)
  • C#

Unity + C#で開発した本作は、私にとって初めてのゲームエンジンの利用であり。それまで経験したことのない規模で、ゲーム状態管理、AI実装、UI設計など、多岐にわたる技術を一度に学び実践する貴重な経験となりました。
